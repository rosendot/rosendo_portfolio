# Certificates Directory

Place your certificate images here.

## Recommended Format

- **File Format:** PNG or JPG
- **Aspect Ratio:** 16:9 or similar
- **Resolution:** At least 1920x1080 for best quality
- **File Size:** Optimize to under 500KB per image

## Naming Convention

Use descriptive names:
- `aws-certified-developer.png`
- `google-cloud-associate.png`
- `react-certification.png`

## Usage

Reference images in `/src/app/certifications/page.tsx` like this:

```typescript
image: "/certificates/your-certificate-name.png"
```

Then uncomment the image display code in the certifications page.
